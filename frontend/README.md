# 젤핏(JellFit) Frontend README
## Frontend 실행 가이드
### 1. 의존성 설치
```bash
cd frontend
yarn install
```
### 2. 개발 서버 실행 (로컬 환경)
```bash
yarn start         
```
### 3. 일렉트론 실행 (데스크탑 테스트용)
```bash
yarn electron start         
```
